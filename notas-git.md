# Notas Git - Aprendizaje

## Comandos basicos aprendidos

git config --global user.name "tu nombre"   → configurar identidad
git init                                     → crear repo local
git status                                   → ver estado archivos
git add .                                    → preparar cambios
git commit -m "mensaje"                      → guardar version
git remote add origin <url>                  → conectar con GitHub
git push -u origin main                      → subir a GitHub
git push                                     → subir cambios futuros
